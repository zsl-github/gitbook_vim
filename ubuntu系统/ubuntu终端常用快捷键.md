终端进行操作的时候，如果来回使用鼠标，会严重影响你的操作。这里就记录下终端常用快捷键，熟练掌握后，距离完全拜托键盘就又近了一步

**Ctrl + a**
光标移动到开始位置

**Ctrl + e**
  
光标移动到末尾

**Ctrl + b**
向前移动光标

**Ctrl + f**
向后移动光标

**Ctrl + k**

删除此处至末尾的所有内容

**Ctrl + u**
删除此处至开始的所有内容

**Ctrl + d**
删除当前字符


**Ctrl + h**
删除当前字符前一个字符

**Ctrl + w**
删除此处到左边的单词

**Ctrl + y**
粘贴由Ctrl+u， Ctrl+d， Ctrl+w删除的单词

**Ctrl + l**

相当于clear，即清屏
