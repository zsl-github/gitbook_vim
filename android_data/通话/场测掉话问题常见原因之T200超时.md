## 查看掉话原因

```
2017 Nov  4  07:10:58.867  [3D]  0x5AC8  GSM DSDS L2 States
Subscription ID = 1
SAPI = SAPI0
L2 State = Release Pending
L2 Event = EV_T200_TIMEOUT
Length:  16
Header:  10 00 C8 5A 3D 81 F5 D8 84 5D DE 00
Payload: 01 00 04 11
```

这种掉话就是因为网络测接收不到上行数据，网络超时，然后发生掉话
