# 背景
今天折腾`tmux`的时候，偶然看到有篇文章说这个`shell`比自带终端要强大的多，故安装来尝试下

# 安装方法

**安装zsh**

`ubuntu`下直接执行命令

```bash
apt-get install zsh
```

**安装oh my zsh**

直接执行命令即可

```bash
wget https://github.com/robbyrussell/oh-my-zsh/raw/master/tools/install.sh -O - | sh
```
