今天遇到一个问题，描述如下

```
姓名中包含Emoji表情符号的联系人保存到SIM卡后不显示联系人姓名
```

原因是因为转码失败

```
11-07 20:50:53.222  2187  2187 D SimPhoneBookAdnRecordCache: Loading all ADN records done
11-07 20:50:53.222  2187  3187 D RILJ    : [7085]> RIL_REQUEST_UPDATE_ADN_RECORD with SimPhoneBookAdnRecord{index =0, name = 😃🙄🙄, number = 137133588890, email count = 0, email = null, ad number count = 0, ad number = null} [SUB1]
11-07 20:50:53.230   857   920 E RILQ    : (1/857): RIL[1][main] qcril_cm_ss_convert_utf8_to_ucs2: Error in converting utf8 string to ucs2

```
说明`SIM`不支持此种编码格式的字符
