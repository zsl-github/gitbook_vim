# 背景介绍
在使用`vim`编辑的过程中，经常会跟对接符打交道，这个插件是用来自动补全对接符的，比如，你编辑插入一个`"`，会自动生成`""`

# 插件安装

直接在`.vimrc`中添加如下内容即可

```vimrc
Bundle 'Raimondi/delimitMate'
```
# 配置

这个插件保持默认就可以了，因为目前就用到它的自动补全对接的作用，所以，剩余的作用以后在探索吧

# 使用
不需要使用什么快捷键，编辑模式的时候，会自动补全的
